# Examples

This folder contains public, reference-grade example workflows for `lobster-workflows`.

- `basic-managed-workflow/`: smallest self-contained managed workflow using `runnerType: 'lobster'`
- `approval-telegram-smoke/`: minimal human-in-the-loop example with a Lobster approval checkpoint and Telegram approvers

These examples are intentionally generic:

- no machine-specific absolute paths
- no real Telegram user IDs
- no user-specific tokens or secrets
